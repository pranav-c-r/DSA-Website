# AVL Tree Visualizer

A beautiful dark-themed AVL Tree visualization tool built with React.

## Features

- Insert nodes with automatic balancing
- Delete nodes with automatic balancing
- Visualize balance factors
- Generate random trees
- Clear the tree
- Responsive design

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000` in your browser

## Color Palette

- Primary: Deep Purple (#6B46C1)
- Secondary: Emerald (#38B2AC)
- Background: Dark Gray (#1A202C)
- Text: Light Gray (#E2E8F0)
